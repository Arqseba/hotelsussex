DEMO MODERNA - HOTEL SUSSEX CÓRDOBA

Archivos incluidos:
- index.html
- styles.css
- script.js

Cómo probar:
1. Descomprimir el ZIP.
2. Abrir index.html con doble clic.
3. Para subir una prueba online, cargar estos archivos en una carpeta del hosting.

Notas:
- La demo usa fotos del sitio actual del Hotel Sussex mediante URLs remotas.
- El formulario de reservas abre WhatsApp con un mensaje prearmado.
- También queda un link al motor actual de reservas externo.
- El mapa está embebido con Google Maps para San Jerónimo 125, Córdoba.

Secciones incluidas:
- Inicio con hero moderno.
- Buscador/consulta de reserva.
- El Hotel.
- Nuestra Historia.
- Habitaciones.
- Galería.
- Sussex Eventos.
- Ubicación con mapa.
- Contacto y WhatsApp flotante.
