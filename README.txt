DEMO WEB HOTEL SUSSEX CÓRDOBA

Archivos incluidos:
- index.html
- styles.css
- script.js

Cómo probar localmente:
1. Descomprimir el ZIP.
2. Abrir index.html con doble clic en el navegador.

Cómo subir a hosting para prueba:
1. Subir index.html, styles.css y script.js a una carpeta pública del hosting.
2. Mantener los tres archivos en la misma carpeta.

Reservas:
- Esta demo no tiene backend ni motor de reservas real.
- El formulario de reserva abre WhatsApp con un mensaje precompletado.
- En una versión final se puede reemplazar por:
  a) un motor de reservas externo embebido,
  b) un link al proveedor actual,
  c) un formulario con envío a email/base de datos.

Imágenes:
- La demo usa imágenes públicas del sitio actual del hotel mediante URL externa.
- Para producción conviene descargar, optimizar y alojar las imágenes en el propio servidor.
